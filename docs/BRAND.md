## Usage

Append the signature block to all documentation files, source headers, and configuration files. This establishes provenance, ownership, and brand continuity across every component of the system.

### README Footer
Add the full signature block at the bottom of every README.md.

### Source Headers
Add a compact-line header to every Python file:
```python
# INTELLITRADE — Drtlemon Elite Tech Conglomerate
# Proprietary. Unauthorized distribution prohibited.
```

### Blueprint Documents
Add the compact footer to all blueprint docs:
```
---

---

*INTELLITRADE — Drtlemon Elite Tech Conglomerate*
*Engineering the asymmetric edge. Systems over outcomes. Mathematics over emotion.*
```
